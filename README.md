# AI-Based Detection of Ghost Infrastructure Using Satellite Data

This project explores how satellite data and AI can be used to detect underutilized or unused urban infrastructure (ghost infrastructure).

## Technologies Used
- Google Earth Engine
- Sentinel-2 Satellite Imagery
- VIIRS Night-Time Lights
- Landsat Time-Series Data
- Unsupervised Machine Learning (K-Means)

## Methodology
1. Built-up area detection using Sentinel-2
2. Human activity analysis using night-time lights
3. Temporal activity analysis using Landsat
4. Infrastructure Utilization Index (IUI) computation
5. AI-based clustering to identify ghost infrastructure
